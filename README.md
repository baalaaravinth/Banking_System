# 🏦 Banking System (Java)

A simple Banking System project developed using Java.

## 📌 Features
- Create Account
- Deposit Money
- Withdraw Money
- Check Balance
- Transaction Handling

## 🛠 Technologies Used
- Java
- OOP Concepts
- File Handling

## 🚀 How to Run
1. Clone the repository
2. Open in your IDE (IntelliJ / Eclipse)
3. Run Main.java
